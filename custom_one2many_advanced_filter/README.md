# One2Many Advanced Filter - Odoo 18

Embedded advanced filters for One2many list views inside Odoo form views.

## Compatibility

This package is adjusted for Odoo 18.0.

## Notes

- Frontend-only module.
- Depends on `web`.
- Adds a filter panel before the One2many list renderer.
- Includes defensive compatibility handling for Odoo 18 ListRenderer props.
